Constraint: Do not give me a "Hello World" example. Give me a production-ready component/structure for the [SPECIFIC_PAGE: e.g., Main Analytics View with charts from Recharts/Tremor].

**4 Pillars for a Standout Project**

1. **Glassmorphism or Bento Layout**  
   "Use a Bento-grid style layout for the dashboard to organize information clearly. Incorporate subtle micro-interactions (hover states, spring animations) using Framer Motion or GSAP. Add neumorphism/glassmorphism cards with backdrop-filter."

2. **State Management & Latency**  
   "Implement Optimistic UI updates so the interface feels instantaneous even before the server responds. Use SWR, React Query, or TanStack Query for smart data fetching, caching, and mutations. Add skeleton loaders."

3. **Efficiency First**  
   "Include a global search bar (CMD+K) that allows users to navigate the entire app via keyboard. Provide 'Bulk Actions' UI for handling multiple data points at once. Support infinite scroll and virtualized lists (react-window)."

4. **Scalability & Security**  
   "Add authentication (NextAuth/Clerk), role-based access control, input validation (Zod), API rate limiting, and monitoring (Sentry). Make it PWA-ready with service workers."
